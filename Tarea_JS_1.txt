Las variables dentro de javascript son una forma de almacenar información que
puede ser cambiada en cualquier momento. Podemos crearla desde cualquier tipo y
modificarla hacia cualquier tipo, como por ejemplo puedo crear una variable con
un valor nùmerico y despues volverla una variable de "string".

Una "string" o cadena, no es más que información de texto, la cual siempre va a
estar definida entre comillas simples o dobles.
Las funciones siempre iniciaran con la palabra function, ademas le daremos un
nombre, si queremos, para que nos describa lo que hace, tambien se agregara un parentesis
donde puede ir el argumento y despues un corchete donde ira nuestro codigo. Las funciones necesitan
variables para hacer algo, como por ejemplo una funciòn puede cambiar el valor
de una variable o realizar alguna operación.

Los argumentos son variables que iran dentro de los parentesis de las funciones a lo que si queremos
agregar mas de un argumento necesitamos, necesitamos separarlos por comas, dentro de los parentesis.
La declaraciòn return, nos sirve para que nos muestre el resultado de lo que se le especifique,
como por ejemplo el resultado de una variable que se contenga alguna operación.

Los booleanos o valores booleanos, son aquellos que solo puede ser por ejemplo verdadero o falso, activado o desactivado,
1 o 0, etcetera. Dentro de Javascript las valores booleanos siempre van a  ser verdadero o falso.
